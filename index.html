Recent Projects

  •

    LA Smart Crime Insights

    EDA, Python (scikit-learn, pandas, numpy, matplotlib, seaborn), Tableau, ETL, Machine Learning

I researched & collected LAPD crime datasets & cleaned them, merged with census data for the team to perform comprehensive insights into crime trends across the LA city & developed interactive statistical dashboard.

  • Brain Tumor MRI Dataset Analysis & Classification

    EDA, Jupyter Notebook, Python (scikit-learn, pandas, numpy, matplotlib, seaborn, PyTorch/TensorFlow for neural network models), Machine Learning

    This project used supervised, unsupervised and neural network models to solve brain tumor MRI images classification problem. I collected the datasets, preprocessed and analyzed them. Subsequently I built the classification models (MLP, CNN ResNET50) for tumor detection and segmentation.

    EDA, Python, R, Linear Regression, Logistic Regression

I conducted EDA and built linear regression models with R to predict game sales.
